◆ flow()

float ZephyrFlowRateSensor::flow ( ) const
inline

Read the most recently polled flow rate value converted to either SCCM or SLPM depending on the sensor type.

Update this value by calling readSensor() before reading.

For SCCM sensors: Flow Applied = Full Scale Flow * [(Digital Output Code/16384) - 0.5]/0.4

For SLPM sensors: Flow Applied = Full Scale Flow * [(Digital Output Code/16384) - 0.1]/0.8

Returns
The flow rate value from the most recent reading in the units of the sensor (SCCM or SLPM)
